# BNPL-Research-Paper
# AF3008 - BNPL & Consumer Spending in Fashion E-Commerce

## Project Overview
This repository contains the data and statistical analysis code for our final research project in Business Research & Data Mining (AF3008). The study investigates how Buy Now Pay Later (BNPL) services influence consumer spending behavior in Pakistan's fashion e-commerce sector.

## Files Included
* `Research Survey.xlsx`: The primary survey dataset (n=250) collected via Google Forms.
* `BNPL_data_analysis.py`: Pandas and Matplotlib script to clean data and generate visualizations.
* `BNPL_statistical_tests.py`: Scipy script running independent t-tests on shopping behaviors.
* `BNPL_demographics_stats.py`: Scipy script running Chi-Square tests on categorical outcomes.

## Execution
All Python scripts were executed in Google Colab using Python 3.11.
